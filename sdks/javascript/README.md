# Architect JavaScript SDK

## Example usage

### Prereqs

* Ensure you have node v20 or newer
* `npm install`

### Environment Setup

* Navigate to your Architect account (https://app.architect.co/user/account) and
create an API key
* Create a copy of `.env.example` at `.env` with keys filled in
